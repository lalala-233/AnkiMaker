## 软件说明

AnkiMaker 是一个有用的软件，可以为 Anki 生成卡片。

但现在，AnkiMaker 只能为中文的古诗词生成卡片，并且只能生成可以被 Anki 导入的 .txt 格式的卡片。

## 前提条件

建议使用最新版本的 Anki 来导入 AnkiMaker 生成的卡片。

如果你使用的是一个过旧的版本（<=2.1.54），那么卡片可能无法正确导入。
